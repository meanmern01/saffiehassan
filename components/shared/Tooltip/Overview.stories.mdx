# Tooltip

Tooltip is used to show an additional text or markup when hovering an element.

Restriction: the direct child of a component should have the ability to have a [react ref](https://reactjs.org/docs/refs-and-the-dom.html).

<Canvas withToolbar>
    <div style={{ height: 110 }}>
        <Tooltip label={<div style={{ padding: 10 }}>I am the description of the element</div>}>
            <span>Hover me</span>
        </Tooltip>
    </div>
</Canvas>

<ComponentProps component={Tooltip}>
    <p>Supports all the default HTML "div" attributes.</p>
</ComponentProps>
