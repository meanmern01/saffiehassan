# ActionWithConfirmation

Delays user action until his explicit confirmation. Child of this component is considered a user action's target.

    <div style={{ height: 200 }}>
        <ActionWithConfirmation

                    <Button variant="dark" data-confirmation="accept">
                        Remove
                    </Button>

                        Keep
                    </Button>
                </>
            }
        >
            <Button variant="dark">Delete ingredient</Button>
        </ActionWithConfirmation>
    </div>
